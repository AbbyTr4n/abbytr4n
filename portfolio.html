1.1.9 Python Project: Build a Bunny!

This project utilizes user input to select the background, color of the bunny (pink, black, white, gray, brown), size of the bunny (small, medium, large), and if the user wants a cute face on the bunny! Iteration and if statements were employed to draw the various shapes and designs for the bunny. There is also a cute pet turtle for the bunny at the end.


1.2.5 Python Project: Turtle Muncher

Users will use the WASD keys to move a turtle around a selected background in their selected difficulty level (easy, hard). The user will have to move their turtle to avoid the bombs and eat the kelp within the time limit. If they succeed, they win. Lists, user input, and defined functions were utilized to have users choose their background and establish game-ending cues.

Scratch Project: Halloween Platformer

Users will use the arrow keys to move their selected character toward the haunted house through various levels within a specific time limit. Along the way, they must avoid pumpkin obstacles and collect candy corns. Hallowen music is present throughout the entire game and the backgrounds change with every level.


2.1.6 Project: A pHishy Fish Tank

Ultimately, we were able to change the code correctly, eliminate unexpected errors in try/except cases, and add recommendations to improve the code.

4.1.4 Project: Understanding Complex Systems in Dining Philosophers

We selected the game Dining Philosophers in Netlogo and analyzed it's systems. The game displayed a relationship between thinking and hunger, had turtles for the philosophers and forks, used randomness to determine how long a philosopher thinks, and can be applied to real-life by questioning how world hunger can impact the intellectual growth of a community.

3.1.6 Project: Rover Home Phone

We deduced that the data from the rover aligns with the Rocky Mountains. After separating the data into four sections, we graphed each. The sound graph stays flunctuating around 75, thus aligning with the low howl of the Rocky Mountains. The gyroscope graph changes between positive and negative values which follow how the Rocky Mountains have varying intensities of constant winds. The lighting graph shows gradual increases and decreases throughout the day, which can be explained by the Rocky Mountain's occasional clouds. Lastly, the temperature graph shows a spike at the end which connects to how the Rocky Mountains spikes in the day time. Therefore, from the graphs we deduced that the data most strongly aligns with the Rocky Mountains.
